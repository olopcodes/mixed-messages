# Mixed Messages

A project from code academy, building an app that displays different answers based on user input.

The app will print a message based on the mood you input.

## How to run the app

The app is a command line app so you have to have node installed.

To start it
`node script.js`

Select your feeling (1, 2, 3) and watch a message appear

- Javascript
